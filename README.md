

# Games Grid Library
This is a growing collection of Grid Images for games on Steam, GOG, etc.

 - The grid is organized in folders according to the game.
 - All images are in JPG or PNG format.
 - The formats are:
	 - Poster (600 × 900)
	 - Hero Banner (1920 × 620)
	 - Banner (1920 × 620)
	 - Logo (1280 × 720)
	 - Icon (512 × 512)
	 - Wallpaper (1920 × 1080)
